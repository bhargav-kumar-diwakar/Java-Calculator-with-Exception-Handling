# Java-Calculator-with-Exception-Handling

## Description
This is a menu-driven Java calculator program that performs addition, subtraction, multiplication, and division. It includes exception handling to prevent crashes when the user enters invalid input or tries to divide by zero.

## Features
- Addition, subtraction, multiplication, and division.
- Handles invalid input (non-numeric characters) using InputMismatchException.
- Handles division by zero using ArithmeticException.
- Continuous menu-driven interface until the user chooses to exit.

## How to Use
1. Run the `Calculator.java` file.
2. Enter a number corresponding to the operation you want:
   - 1 for Addition
   - 2 for Subtraction
   - 3 for Multiplication
   - 4 for Division
   - 5 to Exit
3. Enter two numbers to perform the selected operation.
4. The result will be displayed.
5. Repeat until you choose to exit.

## Requirements
- Java JDK installed
- Terminal or IDE to run Java programs
